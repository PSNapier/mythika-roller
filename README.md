# Mythika Roller

A web-based creature breeding roller for the Mythika universe. Enter two parent creatures (bloodline, genetics, species, rank, stats), optionally select items and pets, then roll to generate offspring results.

## Features

- Parent selection with genetics, species, rank, and stats
- Bloodline/inbreeding checks
- Litter size based on parent ranks and modifiers (e.g. Soul Apocalypse, Fertility Elk)
- Phenotype from genotype (coat colours, markings)
- Skills and runes rolls
- Optional items and pets that affect rolls

## Tech

- Vanilla JavaScript (ES modules)
- Tailwind CSS
- Vitest for tests

## Setup

```bash
npm install
```

## Usage

Open `index.html` in a browser (or serve the project folder locally).
